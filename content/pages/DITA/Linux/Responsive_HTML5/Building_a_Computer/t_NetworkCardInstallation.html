Installing a Network Adapter

Installing a network adapter only takes a minute. If your motherboard does not have one already included, follow these steps to install one.

  1. Put on your anti-static strap or touch the metal computer case before starting.
  2. Find an open PCI slot in your computer.
  3. Remove any I/O shield in place so you will have access to it once the case is closed.
  4. Insert the network card, pushing firmly on one side and then working your way to the other side.
  5. If you have an antenna for wireless that came with the network card, screw it in at the end.
